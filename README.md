# PTIMS
一个使用flask和MySQL的简单的公交信息管理系统
## 功能
1.用户注册登录  
2.地图简单交互  
3.数据导入导出  
--------
## 使用方法  
1.在app/mag.html第321行填入自己的[百度地图api](https://developer.baidu.com/map/index.html)  
2.安装启动docker  
3.运行prework.py生成mysql docker  
4.运行run.py
